ENGLISH: READ ME.

Scenario: A local NGO needs to publicize an animal adoption event, but they only have a simple text document. People aren't reading it because the information lacks hierarchy, links, or images.

Challenge: Create a page that organizes this information in a way that any browser in the world can understand the importance of each sentence.
